# E-commerce Business Analytics (MongoDB)

Анализ международного e-commerce бизнеса с использованием MongoDB и Power BI.

Цель: исследовать продажи, возвраты, сезонность и различия между странами для выявления точек роста бизнеса и повышения эффективности продаж.

Данные:
https://www.kaggle.com/datasets/carrie1/ecommerce-data

## Инструменты
- MongoDB
- Docker (развертывание MongoDB)
- Python
- Power BI

## Результаты
- проведён анализ выручки по странам
- исследована структура заказов и средний чек
- выполнен анализ возвратов товаров
- изучена сезонность продаж
- построены интерактивные дашборды для бизнес-анализа

## Ключевые инсайты
- около 84% выручки приходится на Великобританию → высокая зависимость бизнеса от одного рынка
- выявлены страны с высокой долей возвратов при относительно небольшой выручке → потенциальная зона оптимизации логистики и ассортимента
- отдельные рынки демонстрируют потенциал роста при низкой текущей доле выручки
